Консольний застосунок розроблений мовою програмування C# на платформі .NET та призначений для автоматизації базових процесів книжкового магазину.

Програма забезпечує роботу з даними користувачів, товарів та продажів із використанням файлового зберігання у форматі CSV. 
Всі дані зчитуються та зберігаються під час виконання програми, що дозволяє змінювати інформацію без перекомпіляції застосунку.

У межах проєкту реалізовано основні операції обробки даних, зокрема перегляд інформації, пошук та базові CRUD-операції.
Користувацька взаємодія здійснюється через текстове меню в консольному режимі.

Проєкт створений у рамках навчальної практики з дисципліни програмування та демонструє навички роботи з мовою C#, файловою системою та алгоритмами обробки даних.
